# Example configuration files

Starter files for PermissionsExPlus. Copy to your server's `plugins/PermissionsEx/` directory.

| File | Destination |
|------|-------------|
| `config.yml` | `plugins/PermissionsEx/config.yml` |
| `permissions.yml` | `plugins/PermissionsEx/permissions.yml` |

Documented on the site: [Example configuration files](https://permissionsexplus.rono.dev/guides/example-configs).
