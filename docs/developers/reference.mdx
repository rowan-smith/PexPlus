## Generate locally

**%%site.version%%** (PermissionsExPlus):

```bash
mvn -pl api/permissionsex-api,legacy-api/permissionsex-legacy-api javadoc:javadoc -am -Ddoclint=none
```

**Classic 1.23.x**:

```bash
./scripts/build-classic-javadoc.sh STABLE-1.23.4 1.23.4
```
